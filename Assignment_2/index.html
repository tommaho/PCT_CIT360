Assignment Description - Most Varied Substring (click to expand)

You are given a string S and an integer L. Your task is to find the substring of length L in S that has the greatest number of distinct characters.

For instance, given

  • S = “TTRTTSSSSTSTSTS”
  • L = 4

The output should be:

  • “RTSS” 3

If there are multiple substrings with the same property, cite the first one.

Your program should read pairs of strings and numbers from a file called “string_assignment.dat” and for each pair, output (in the above format) the result to the standard output.

Input file format:

An integer number in the first line of the text indicates the number of input pairs

Strings and numbers are placed on separate lines in consecutive order.
